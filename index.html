How to add yourself (students)

  1. Click Add file → Create new file.
  2. Name it like _students/first-last.md (use dashes, all lowercase).
  3. Paste the template from this page (scroll down) and fill in your details.
  4. Click Commit changes (create a new branch) → Propose changes → Open pull request.
  5. Wait for approval. Your name will appear below after it’s merged.
